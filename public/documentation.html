Usage

Initialize the player with the following code:


const player = new CustomPlayer('#video-container', {
    src: 'assets/sample.mp4',
    type: 'video/mp4',
    captions: [
        { src: 'assets/captions.vtt', kind: 'subtitles', srclang: 'en', label: 'English', default: true },
    ],
    theme: 'light',
});

API

Explore available options:

  • src: Path to the video file.
  • type: MIME type of the video (e.g., `video/mp4`).
  • captions: Array of caption tracks.
  • theme: `light` or `dark` theme.
